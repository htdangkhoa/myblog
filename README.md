# myblog

[![Greenkeeper badge](https://badges.greenkeeper.io/htdangkhoa/myblog.svg)](https://greenkeeper.io/)

## Requirement

- [NodeJS v10](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)

## Getting Started
```bash
$ npm install
```

### For development

```bash
$ npm run dev
```

### For production
```bash
$ npm start
```
